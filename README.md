Chinese word segment use Bi-LSTM and CRF 

#### requirements:
- **python3.5+**
- **tensorflow-gpu>=1.8**
- **pandas**
- **numpy**
- **scikit-learn**

Usage:

1. run `data_processing.py` to convert the original corpus to dataset
2. `cd src`find all the models
3. go into a model directory and run `python train.py`
